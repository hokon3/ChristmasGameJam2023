Name:
  Christmas Tree
Version:
  1.0
Description:
  Christmas tree animated tileset created by bleutailfly for Stendhal.
  Based on b_o's Pine Tree Tiles.
Details:
  - Orientation: orthogonal
  - Dimensions: Designed for 32x32 tiled maps, but should work for others.
  - Animation:
    - Animated preview frame delay: F1: 180ms, F2: 160ms, F3: 320ms, F4: 160ms
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
